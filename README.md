# GONDOLA-ON
Atelier html pour la classe Gondola

Etape1: créer son projet dans github
Etape2: cloner le projet sur mon ordinateur
Etape3: Ouvrir mon projet cloné avec vscode
Etape4: Faire la structure de mon projet
Etape5: Envoyer mes mise a jour sur la plateforme Github
Pour envoyer sur la plateforme github voici les
commandes a taper:
    1: git add .
    2: git commit -m "structure de mon projet"
    3: git push
NB: Les trois commandes ci-dessus sont les commandes que nous allons taper
tout au long de notre projet si nous voulons faire des mise a jour
